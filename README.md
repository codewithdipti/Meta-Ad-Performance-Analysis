##  Problem Statement
Businesses running advertisements on Meta platforms (Facebook and Instagram) generate large volumes of data related to impressions, clicks, engagement, and conversions. However, this data is often scattered and difficult to analyze effectively. Marketing teams struggle to identify high-performing campaigns, understand audience behavior, and optimize ad spend.

This project aims to build an interactive dashboard that consolidates Meta ad data and provides actionable insights to improve campaign performance and return on investment (ROI).
##  Project Objective
- Analyze Meta (Facebook & Instagram) advertising data
- Track key performance indicators (KPIs)
- Understand audience demographics and behavior
- Evaluate ad type effectiveness
- Identify trends across time and geography
- Support data-driven marketing decisions
##  Tools & Technologies

- **CSV Files**

- **DAX Measures**
- **Data Visualization**
- **Business Analytics**
##  Dataset Description
The project uses the following datasets:

| File Name | Description |
 `users.csv` - User demographic information 
  `campaigns.csv` -Campaign-level details 
 `ads.csv` -Ad creatives and ad types 
 `ad_events.csv` - Events such as impressions, clicks, engagement, purchases 

##  Key Performance Indicators (KPIs)
- Impressions
- Clicks
- Click Through Rate (CTR)
- Engagements
- Engagement Rate
- Purchases
- Conversion Rate
- Purchase Rate
- Total Budget
- Average Budget per Campaign
## Dashboard Features
- **KPI Cards** for overall performance tracking
- **Gender & Age Analysis** to understand audience distribution
- **Geographic Analysis** using world map visualization
- **Hourly & Weekly Trends** for time-based insights
- **Ad Type Performance** comparison (Image, Video, Carousel, Stories)
- **Platform Filter** (Facebook / Instagram)
- **Dynamic Slicers** for campaign, interests, and metrics
